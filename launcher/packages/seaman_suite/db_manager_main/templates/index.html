Seaman PC Database System

UDB files store game state encrypted with IDEA cipher (ECB mode, 128-bit key).

Values are XOR 0xFF encoded Shift-JIS text with 15-byte timestamps.

DPC files define the variable path templates (932 paths in SeamanPC.dpc alone).
