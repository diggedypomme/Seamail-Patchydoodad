# Imported Tool Bundles

This folder is for intact imported projects that the launcher may start or link to, but which should keep
their own internal folder structure.

Current bundle:

- `seaman_suite/`
  - full copied tool set and `project_summary.md`
- `position_tracker_main/`
  - standalone live tracker package with templates, static assets, and movement visualisation files

Use `launcher/scripts/` for thin launcher wrappers, not for flattening these packages.
